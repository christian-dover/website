Scholarship Project

This project takes input such as name, GPA, and scholarship amount, and prints it back in a nicely formatted way. I learned how to control decimal places, align text, and creatively format output when spacing is limited. 

#include <iostream>                                     
#include <string>                                       
#include <iomanip>                                      

using namespace std;

int main()
{
  string name;
  double gpa;
  int scholarship;
  // get input from the user (name,gpa,and scholarship)
  cout << "what is you name?" << endl;
  cin >> name;
  cout << "what is your gpa?" << endl;
  cin >> gpa;
  cout << "how much is your scholarship?" << endl;
  cin >> scholarship;

  cout << fixed << showpoint << setprecision(2);

  cout << left;

  cout << setw(10) << name << setw(7) << gpa
    << setw(8) << scholarship << endl;

  // set in *,#,@,and ^ to get user attention and shows user inputs from last questions
  cout << setfill('*');
  cout << setw(10) << name << setw(7) << gpa
    << setw(8) << scholarship << endl;

  cout << setw(10) << name << setfill('#')
    << setw(7) << gpa
    << setw(8) << scholarship << endl;

  cout << setw(10) << setfill('@') << name
    << setw(7) << setfill('#') << gpa
    << setw(8) << setfill('^') << scholarship
    << endl;

  cout << right;
  cout << setfill(' ');
  cout << setw(10) << name << setw(7) << gpa
    << setw(8) << scholarship << endl;

  return 0;
}

Deans List

This program works similarly to the scholarship project but checks whether the user made the Dean’s List or not. I learned how to let the program analyze user input against a set of rules and return different responses based on that input. 

#include <iostream>
using namespace std;
// gets user input from the menu that the user is shown
int main()
{
  int choice;
  int score;
  double GPA;
  do {
    cout << endl;
    cout << "please enter a option\n";
    cout << "1) Enter your test score, I'll tell you the grade. \n";
    cout << "2) Enter your GPA to see if you made the Dean's List. \n";
    cout << "3) exit \n";
    // make sure the user is choosing a choice and not putting in random nonsense
    while (!(cin >> choice))
    {
      cin.clear();
      cin.ignore(100, '\n');
      cout << "please choose one of the options" << endl;
    }
    // from user input give the apporiote output from choice given
    switch (choice)
    {
    case 1:
      cout << "Enter you test score: \n"; 
      while (!(cin >> score))// make sure that puts in number and not just nonsenese
      {
        cin.clear();
        cin.ignore(100, '\n');
        cout << "please list you actual test score in numbers 0 - 100" << endl;
      }
      if (score >= 90 && score <= 100)
        cout << "you have an A\n";
      if (score >= 80 && score <= 89)
        cout << "you have got a B\n";
      if (score >= 70 && score <= 79)
        cout << "you got a C \n";
      if (score >= 60 && score <= 69)
        cout << "you got a D \n";
      if (score <= 59)
        cout << "you got a F\n";
      break;
    case 2:
      cout << "did you make the deans list? Enter you GPA: \n";
      while (!(cin >> GPA || GPA > 0 && GPA <= 4.0))
      {
        cin.clear();
        cin.ignore(100, '\n');
        cout << "make sure this is a number not a letter" << endl;
      }
      if (GPA >= 3.9 && GPA <= 4.0)
        cout << "you have made the deans list congratulations!\n";
      else
        cout << "i'm sorry but you need a 3.9 GPA or better\n";
      break;
    case 3:
      break;
    // shows this option if no choice made on menu is correct
    default:
      cout << "invalid input choose one of the three options\n";
    }
  } while (choice != 3);
  return 0;
}

Linux was the second environment I learned. I gained a lot of experience with how it’s used in the business world—especially for servers, storage, and automation.

Large Volume Manager (LVM)

When I started working with Linux, I noticed how often it’s used to manage servers and large amounts of data. I decided to learn how to split and manage storage using LVM.

I chose Debian for this project because it’s a stable and popular choice for server-related tasks.

To work with storage more flexibly, I used LVM2. It allows me to create and manipulate logical volumes on top of physical storage.

Once LVM is available, it lists different commands and options for managing partitions. Here I started by changing the partition type.

After choosing to change the partition type, I answered a few questions about how I wanted to use it. I selected the Linux LVM type with a FAT32 format for this setup.

Once the configuration is complete, the system asks what you want to change the partition into. I changed it into a Linux LVM partition.

After setting up a usable LVM platform, I allocated storage. In this example, I gave the system 5GB of storage to work with.

As I continued, I realized I needed more groups of storage, so I created a few more LVM partitions.

Since I had multiple LVM partitions, I grouped them using a volume group (VG). I named it LVMVolGroup.

After creating the volume group, all the LVM partitions appeared under that group, making it easier to manage them together.

With the group set up, I created logical volumes for specific purposes: www, projects, and db.

Next, I created directories with the same names (www, projects, and db) so I could mount the logical volumes there.
