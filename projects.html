Large Volume Manager (LVM)

When I started working with Linux, I noticed that a lot of it is used for servers and storage, especially breaking data into manageable pieces. I decided to explore that by using LVM.

I decided to use Debian for this work since it’s a robust operating system for this kind of task.

To work with storage properly, I needed a tool to manage it. I chose lvm2 so I could create and manipulate storage for this project.

Once you start up LVM, it gives a list of commands for working with storage. Here I chose to change the partition type.

After choosing to change the partition type, LVM asks a few questions about how you want to use the partition. I chose the xixux usr option and a FAT32 data format.

Once I finished configuring how I wanted to work with the partition, it asked what I wanted to turn it into. I chose to turn it into a Linux LVM partition.

After creating a workable platform for LVM, I had to give the system storage to work with. I allocated 5 GB of storage.

As I worked through the steps, I realized I needed more storage groups, so I made a few more LVM partitions.

Since I created so many LVM partitions, I needed to put them together into a group. Using the vgcreate command, I created a volume group called LVMVolGroup.

After creating the volume group, all the LVMs I created were listed under that group.

With everything under one tag, I created logical volumes named www, projects, and db.

Then I created directories named www, projects, and db to use as mount points.

Once the directories were created, I mounted the LVM volumes into the system using the mount command and placed them in their respective directories.

import rsa
from cryptography.fernet import Fernet
class key_gen:
    def create_keys(search_term):
        # create the symmetric key
        key = Fernet.generate_key()

        # write the symmetric key to a file
        k = open(search_term + 'symmetric.key','wb')
        k.write(key)
        k.close()

        # create the pub & private keys
        (pubkey,privkey)=rsa.newkeys(2048)

        # write the public key to a file
        pukey = open(search_term + 'public.key','wb')
        pukey.write(pubkey.save_pkcs1('PEM'))
        pukey.close()

        # write the private key to a file
        prkey = open(search_term + 'priv.key','wb')
        prkey.write(privkey.save_pkcs1('PEM'))
        prkey.close()
